# bi-atendimento-mapa
Mapa de cobertura por promotor (farmácias) com Colab/Pandas na ingestão e limpeza, dicionário editável no Google Sheets, VIEW canônica no BigQuery e Looker Studio na visualização; contagem única via LojaPadrao, filtros geográficos is not null e página de Qualidade do Dado. Arquitetura simples, governança clara e métricas estáveis.
